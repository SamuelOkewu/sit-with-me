# How to Create Your og-image.jpg

This file needs to be created and saved as `og-image.jpg` in this `/public` folder.

## Requirements:
- Size: 1200 x 630 pixels (EXACTLY)
- Format: JPG or PNG
- Max file size: 300KB

## What to include:
- Your brand name "Sit With Me" in your serif font
- A beautiful lifestyle/fashion image as background
- Keep text legible with a dark overlay if needed

## Free tools to create it:
1. **Canva** (recommended): canva.com → Create design → Custom size 1200x630
2. **Figma**: figma.com → New frame → Custom 1200x630

## Why it matters:
This image appears whenever someone shares your site link on:
- WhatsApp
- Facebook
- Twitter/X
- LinkedIn
